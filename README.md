# Netflix Clone

A Netflix-inspired frontend streaming UI built with React, TypeScript, Tailwind CSS, Vite, and TMDB API.

## Features

- Netflix-style homepage
- Dynamic hero banner
- TMDB API integration
- Search movies
- Movie detail page
- Watch page
- Trailer player
- Hover preview cards
- Continue Watching
- My List
- Profile selector
- Profile-specific saved state
- Responsive navbar and layout
- Intro splash screen

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide React
- TMDB API
